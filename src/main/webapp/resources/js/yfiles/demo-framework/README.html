Support files for yFiles for HTML programming samples and tutorials

This directory contains support files that are used by all programming samples and tutorials.

The following files are included:
Name Description
yfiles.demo.js A precompiled version of the yFiles demo application framework which provides a common look and feel to all programming samples.
require.js A sample implementation of the AMD loader mechanism that is used by our demos. The mechanism is described in the Getting Started chapter of the yFiles for HTML Developer's Guide.
